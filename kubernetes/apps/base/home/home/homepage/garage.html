⚙️ Architecture

🔑 Replication & Consistency

  • • Replication factor: 2 (read quorum=2)
  • • Consistency mode: consistent — prevents restic AEAD corruption
  • • Layout version v209 — auto-applied via GarageCluster CR

🚪 common-ingress (Inbound)

  • • 3 replicas per cluster — Tailscale operator-managed
  • • Garage storage + gateway pods accept traffic via this group
  • • 8–9 TS services per cluster with annotation: proxy-group: common-ingress
  • • Migrated from standalone TS pods to proxy group (Jun 9)

📤 common-egress (Outbound RPC)

  • • 3 replicas per cluster — outbound Tailscale connections
  • • Each cluster runs all 3 regions' gateway RPC services (13 total)
  • • Template: {zone}-gw-{ordinal}.keiretsu.ts.net:3901
  • • Cross-region RPC flow: Garage → common-egress → VPN → common-ingress → peer
